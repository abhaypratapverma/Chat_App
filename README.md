for starting the project follow these steps

step 1 open first terminal for frontend server and hit npm run dev
step 2 open second terminal for backend chat service and hit npm run dev
step 3 open third terminal for backend mail service and hit npm run dev  
step 4 open fourth terminal for backend user service and hit npm run dev

NOTE\* here make sure you have setup your docker and rabbitMq and added you current ip to mongodb atlas
